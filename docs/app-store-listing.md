## 説明（4000文字以内）

```
ゴルフ感覚メモは、初心者〜100切りを目指す方のためのシンプルな練習・ラウンドメモアプリです。

■ こんな方におすすめ
・良かった感覚やミスの原因を、次のラウンドまで忘れたくない
・スコア管理より「感覚と改善」に焦点を当てたい
・手軽に記録を残したい

■ 主な機能
・練習 / ラウンドの記録（良かった感覚・ミスの原因・次回試すこと）
・クラブ別タグ（ドライバー、ウッド、アイアン番手、アプローチなど）
・種別・タグで一覧フィルタ
・記録の編集・削除
・端末内に保存（オフラインで利用可能）
・YouTubeで「直し方」を検索（記録内容から検索ワードを自動作成）

■ データについて
記録はお使いの端末内にのみ保存されます。開発者のサーバーへ送信しません。

■ お問い合わせ
golfrecord.app@gmail.com
```
