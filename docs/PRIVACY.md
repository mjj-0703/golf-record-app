# プライバシーポリシー — ゴルフ感覚メモ

最終更新: 2026年6月

## 概要

「ゴルフ感覚メモ」（以下、本アプリ）は、ユーザーのゴルフ練習・ラウンドに関するメモを **端末内にのみ** 保存するアプリです。

## 収集する情報

本アプリは、開発者のサーバーへ個人情報を **送信しません**。

端末内（`shared_preferences`）に保存されるデータ:

- 記録の日付・種別（練習 / ラウンド）
- 良かった感覚、ミスの原因、次回試すこと、メモ
- 選択したクラブ・タグ

## 外部サービス

「YouTubeで直し方を探す」機能利用時、ユーザーの操作により **YouTube / ブラウザ** が起動し、検索クエリが外部アプリに渡されます。本アプリは YouTube Data API を使用せず、動画内容や Google アカウント情報を取得しません。

## データの削除

アプリ内の記録削除、またはアプリのアンインストールにより、端末内データは削除されます。
